# Moderation operations

## Report reasons

Use: harassment, misinformation presented as sourced fact, uncited copyrighted text, spam, off-topic, impersonation, personal information, or other.

## Review sequence

1. Confirm the report concerns a community post rather than disagreement with a source edition.
2. Preserve the original in the revision/audit trail.
3. Record the rule, evidence, action, scope, and duration.
4. Prefer the least severe action that protects the community.
5. Record appeal status when an appeal is submitted.

## Available actions

- Moderator note
- Post hide or removal
- Thread lock, pin, or archive
- Warning
- Temporary posting restriction
- Temporary suspension
- Permanent ban

Do not alter source text through moderation tools. Text corrections use the correction and revision workflow.
