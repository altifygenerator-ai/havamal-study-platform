# The Hávamál Archive brand system

## Identity

**Name:** The Hávamál Archive  
**Descriptor:** Texts, translations, notes, and reader discussion.  
**Guiding line:** Read the text. Compare the translations. Form your own understanding.

The interface should feel like a worn manuscript under active study: old but usable, serious but not institutional, Pagan without occult-store styling, and historical without pretending to be a medieval artifact.

## Palette

- Soot frame: `#181411`
- Deep soot: `#100d0b`
- Charred wood: `#2d251f`
- Main vellum: `#d2bd8f`
- Light vellum: `#e5d6b4`
- Pale folio: `#eee3cb`
- Brown-black ink: `#211a14`
- Muted manuscript ink: `#665846`
- Rubric red: `#7b3027`
- Worn leather rule: `#8a7251`
- Old-paper text: `#e7dcc5`

Rubric red is functional: section labels, selected editions, active index entries, source warnings, and small folio marks. It is not a bright marketing CTA.

## Typography

The CSS stack prefers Junicode for the wordmark, headings, stanza poetry, and Old Norse, and Source Serif 4 for reading and interface copy. Neither font file is redistributed with the repository. Install or self-host them only under their own licenses, or retain the supplied system-serif fallbacks.

No fake runic type, blackletter interface copy, generic Inter dashboard styling, or decorative pseudo-Norse lettering.

## Composition

- Archive-index navigation rather than pills.
- Vellum reading leaves set inside a soot-dark frame.
- Thin rules, marginal notes, folio labels, asymmetric columns, and restrained paper grain.
- Rectangular or underlined controls rather than glossy rounded buttons.
- Source text remains the brightest and strongest visual object.
- Reader discussion appears as a later, clearly separated leaf.

## Prohibited visual shortcuts

No horned helmets, glowing runes, random bindrunes, axes as filler, ravens used as a generic logo, fantasy warriors, parchment scroll clichés, burned edges, fake wax seals, blue SaaS cards, gradients, or invented manuscript decoration.
